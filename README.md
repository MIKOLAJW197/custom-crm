## Description

Custom CRM project made for univeristy.

## Installation

### Database
Need to install Docker first!

```bash
$ cd backend
$ docker-compose up -d
```

### Backend
```bash
$ cd backend
$ npm install
```

### Frontend
```bash
$ cd frontend
$ npm install
```

## Running the app

### Frontend
```bash
$ cd frontend
$ npm run start
```

Then navigate to localhost:4200

### Backend
```bash
$ cd backend
$ npm run start
```
